Using modern typeface Commissioner for clean sans-serif UI, and JetBrains Mono for tabular numbers, IDs, and code blocks. No default browser fallbacks.

Body Paragraph (text-secondary)

Our standard body text is sized dynamically with line height of 1.75. It utilizes a comfortable slate-zinc gray shade to prevent user fatigue while reading long-form logs, database lists, and invoices.
