Bulk Upload Products

1. Download the Template

Select a business category to generate a tailored CSV template. The downloaded file will only contain columns relevant to the selected business type, alongside standard product info.

• Uploading Products with Variants ▼
  • For products with variants, upload the first variant using the generic product slug. Leave the Parent SKU field blank. Fill in the Variant Name and Variant Value fields.
  • For remaining variants, use the SKU of the first uploaded product as the Parent SKU. Fill in the SKU, Slug (variant-specific), Variant Name, and Variant Value fields.
  • Ensure all variants reference each other in their Related SKUs field.

2. Upload Products CSV

Select your completed CSV file. This tool will automatically parse the data, calculate prices, extract metadata, and insert the records into Supabase.
