Step-by-Step Email Domain Connection Guide

Connecting your business domain ensures that user verification links, workspace invites, and HR alerts are sent from your official brand address instead of falling back to system defaults.

1
Sign up for a Resend account

Go to resend.com and sign up. The free tier supports sending up to 3,000 emails per month.

2
Add and verify your domain

In your Resend dashboard, select Domains → Add Domain, and type your custom domain name (e.g. mycompany.com). Add the provided DNS MX/TXT records to your domain provider (e.g., GoDaddy, Namecheap, Cloudflare) to verify it.

3
Generate an API Key

In the Resend dashboard, go to API Keys → Create API Key. Set the permission to Sending Access, copy the key (starting with re_), and save it in a safe place.

4
Save integration settings in BrightKey

Paste the API Key in the field above. Set your sender email address (e.g., HR Alerts <onboarding@mycompany.com>) using your verified domain, and click Save Settings.
